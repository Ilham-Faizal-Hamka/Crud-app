## Prerequisite
- [Download](https://nodejs.org/en/download/) and install Node.js LTS version 18.12.1 <br/>
- Clone this project and open it using your favorite code editor.<br/>

## Setting up project
Install required dependencies:<br/>
```
 npm install
```
## Run the program
### Client : 
```
    cd client 
    npm start
```
### Api
```
    cd api
    npm start
```
