## Prerequisite
Download and install Node.js LTS version 18.12.1
## Setting up project
Install required dependencies:<br>
```
 npm install
```
## Run the program
### Client : 
```
    cd client 
    npm start
```
### Api
```
    cd api
    npm start
```
